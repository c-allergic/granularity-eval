# Granularity-Eval

Multi-granularity caption evaluation experiment. Demonstrates that existing image caption metrics (CAPTURE, CompreCap, etc.) have systematic granularity bias — failing to align with human judgments on short but accurate captions.

## Setup

```bash
pip install -r requirements.txt
```

## Data

Place images in `data/images/{category}/`:
- `natural/` — 100 natural scene images from MSCOCO
- `doc_poster/` — 50 document/poster images
- `art/` — 50 art/oil painting images

Place reference captions in `data/references/{image_id}.json`:
```json
["ref caption 1", "ref caption 2", ...]
```

## Pipeline

```bash
# 1. Generate captions (InternVL2-8B on GPU, GPT-4o via API)
python generate.py --internvl    # run on L40
python generate.py --gpt4o       # requires OPENAI_API_KEY

# 2. Run evaluation metrics
python compute_metrics.py ngram spice clipscore capture llava_critic
python compute_metrics.py capsbench  # requires existing CapsBench setup

# 3. Human evaluation (after collecting annotations)
python human_eval/build_hits.py      # generate HITs
python human_eval/aggregate.py       # aggregate results

# 4. Analysis
python analysis.py
```

## Metrics

| Type | Metric | Reference |
|------|--------|-----------|
| n-gram | BLEU-4, METEOR | ref-based |
| Scene graph | SPICE, CAPTURE, CompreCap | ref-based |
| Embedding | CLIPScore | ref-free |
| VQA-based | CapsBench | ref-free |
| VLM Judge | LLaVA-Critic-7B | ref-free |

## Experiment Design

- 200 images × 3 granularities (coarse/medium/detailed) × 3 accuracy types (accurate/incomplete/inaccurate)
- 9 captions per image, generated by 2 models (InternVL2-8B + GPT-4o)
- Human evaluation: within-grain ranking + cross-grain pairwise comparison
- Meta-evaluation: Kendall τ, Pearson r, Williams test
